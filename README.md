DeepSkill - Modern E-Learning Platform
DeepSkill is a premium web-based e-learning platform built with  React  and  Styled Components . It offers a seamless, interactive experience for students to browse, enroll in, and manage their professional courses.

✨ Core Features

-  📊 Interactive Dashboard : A personalized area to track enrolled courses,    progress, and learning time.
-  🗑️ Course Management : Easy unenrollment and enrollment flow directly from the dashboard.
-  👤 User Profile : Customizable user profiles with bio and personal details.
-  🎓 Comprehensive Courses : Dedicated pages for Graphic Design, Full Stack (Laravel/React), and WordPress Mastery.
-  📱 Responsive Design : Fully optimized for mobile, tablet, and desktop views.
-  🎨 Premium UI/UX :
    - Smooth animations with  Framer Motion .
    - Stylish  Glassmorphism  theme.
    - Interactive  Custom Cursor .
    - Scroll-reveal effects for engaging content delivery.

  🛠️ Tech Stack

-  Frontend : React 19+
-  Styling : Styled-Components
-  Animations : Framer Motion
-  Icons : React Icons (Fa, Fi)
-  Routing : React Router DOM (v7)
-  Carousels : React Slick / Slick Carousel

  🚀 Getting Started

   Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/)

   Installation

1.  Clone the repository :
   ```bash
   git clone <repository-url>
   cd deepskill-web
   ```

2.  Install dependencies :
   ```bash
   npm install
   ```

3.  Start the development server :
   ```bash
   npm start
   ```

4.  Build for production :
   ```bash
   npm run build
   ```

  📁 Project Structure

- `/src/components`: Reusable UI elements (Buttons, Hero sections, etc.)
- `/src/context`: State management (AuthContext)
- `/src/assets`: Images, SVGs, and static assets
- `/src/pages`: Main application views (Dashboard, Profile, Course Pages)
